Frequently Asked Questions

How I Can Create My Contest?

Simply go to the Create Contest page, fill out the form ✅, then click on the All Contests button in the first section. You can view your created contest and, with one click, 🗑️ delete it if needed. Happy contesting! 🎯

Can I Make Comment in Discussion Forum Without login ?

You can only view comments without logging in. After logging in ✅, you can make comments, ✏️ edit your own, and 🗑️ delete them.

How I Can Track My Submitted Coding Question Details?

Simply submit your code ✅, then go to the Profile page to see your pie chart 📊. For full details, visit the Dashboard page via the bottom button on your profile to view all your submitted coding questions.

Can I add Question in Practice Section?

You can't set practice questions, but you can send them via email 📧 to admin@gmail.com.
